Named continuous groups
=======================

.. module:: +replab

RepLAB provides constructions of standard continuous groups.

* `.T`: Describes the standard torus group of given rank.
* `.UnitaryGroup`: Describes the group of unitary matrices (shorthand `.U`).
* `.SU`: Describes the group of special unitary matrices.
* `.Orthogonalgroup`: Describes the group of orthogonal matrices (shorthand `.O`).
* `.SO`: Describes the group of special orthogonal matrices.
* `.Sp`: Describes the compact symplectic group.
