Named groupsÂ¶

RepLAB provides constructions of standard groups.

  • AlternatingGroup: Describes the alternating group on a given domain.

  • CyclicGroup: Describes the cyclic group on a given domain.

  • DihedralGroup: Describes the dihedral group on a given domain.

  • SymmetricGroup: Describes the symmetric group on a given domain (shorthand S).

  • SignedSymmetricGroup: Describes the signed symmetric group.

  • QuaternionGroup: Describes the quaternion group of order 8.

  • UnitaryGroup: Describes the group of unitary matrices (shorthand U).

  • Orthogonalgroup: Describes the group of orthogonal matrices.
