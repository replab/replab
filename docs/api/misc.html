Miscellaneous functions/classes¶

  • Partition: describes an unordered partition of the integers \(\{1,...,n\}\),

  • settings: various global settings such as tolerances,

  • isNonZeroMatrix, tests whether a matrix is nonzero up to a given tolerance using the 2-norm (singular value), with accelerations provided by matrix norm inequalities and cheap norms.

static connectedComponents(adjacencyMatrix)¶

Given an adjacency matrix adj, returns the sets of vertices corresponding to connected components

For adj = [0 0 1; 0 0 0; 1 0 0], it returns the partition {[1 3] [2]}

static connectedComponentsFromEdges(edges, n)¶

Given list of edges, returns the sets of vertices corresponding to connected components

For edges = [1 3] and n = 3, it returns the partition {[1 3] [2]}

static permutationsOrbits(permutations)¶
