As an open-source project, RepLAB relies on a community of users to keep getting better and developping in relevant directions. The project is generously hosted on GitHub. The easiest way to contribute is by sharing a suggestion or reporting an issue directly online. Anyone with a github account can do that (an account can be easily created). Every contribution is valued.
